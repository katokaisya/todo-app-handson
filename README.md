# TodoApp-Handson

シンプルなタスク管理Webアプリケーションです。
ブラウザだけで動作し、サーバー不要でタスクの追加・完了・削除が行えます。

---

## 📋 機能

- タスクの追加
- タスクの完了チェック
- タスクの削除
- ページを閉じてもタスクが保持される（LocalStorage）

---

## 🛠 技術スタック

| 技術 | 用途 |
|------|------|
| **HTML** | 画面構造のマークアップ |
| **CSS** | スタイリング・レイアウト |
| **JavaScript** | タスクの追加・削除・状態管理などのロジック |
| **LocalStorage** | タスクデータのブラウザ内永続化 |

> フレームワーク・ライブラリ・サーバー不要。ブラウザのみで動作します。

---

## 🚀 使い方

1. `index.html` をブラウザで開く
2. 入力欄にタスク名を入力して「追加」ボタンをクリック
3. タスク一覧に追加されます
4. チェックボックスで完了状態を切り替え
5. 削除ボタンでタスクを削除
